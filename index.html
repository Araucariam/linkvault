sync Guide de synchronisation

devices Méthode 1 : Export/Import Manuel

  1. Sur votre appareil source → Paramètres → Exporter JSON
  2. Transférez le fichier (email, cloud, câble...)
  3. Sur l'appareil cible → Paramètres → Importer

cloud Méthode 2 : Cloud Personnel

  1. Hébergez l'app sur votre propre serveur (Apache, Nginx)
  2. Utilisez une base de données partagée (Firebase, Supabase)
  3. Tous vos appareils accèdent à la même URL

share Méthode 3 : Share Target Android

  1. Installez LinkVault depuis Chrome Android
  2. Sur n'importe quelle page web → Partager → LinkVault
  3. Le lien est ajouté automatiquement !

computer Méthode 4 : Sur PC

  1. Ouvrez l'app dans Chrome/Edge
  2. Menu navigateur → Installer LinkVault
  3. Utilisez l'export/import pour synchroniser
